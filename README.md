# Weather App

![Weather App](readme_archives/weather_app_gif.gif)[readme_archives/weather_app_gif.gif]

This repository contains a weather application developed using Flutter and Dart for Android and iOS. The application utilizes the Provider package for state management, SharedPreferences for local storage, HTTP for accessing the OpenWeather API for weather and geocoding information, and Google Fonts for beautiful typography.

## Features

* Display of current weather, including temperature, weather description, humidity, and wind speed.
* Option of change theme color in runtime.
* Search based on manual search.
* Storage of last searched location for quick access.

## Prerequisites

* Flutter SDK: [Flutter Installation](https://flutter.dev/docs/get-started/install)
* Android/iOS device or emulator for running the application
* OpenWeather account to obtain an API key: [OpenWeather](https://home.openweathermap.org/users/sign_up)

## Used Libraries

* [provider](https://pub.dev/packages/provider): For state management.
* [shared_preferences](https://pub.dev/packages/shared_preferences): For local storage of the last searched location.
* [http](https://pub.dev/packages/http): For accessing the OpenWeather API for weather and geocoding information.
* [google_fonts](https://pub.dev/packages/google_fonts): For beautiful typography in the application.
